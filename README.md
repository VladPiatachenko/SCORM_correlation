# SCORM-тренажер: залежності між змінними

Інтерактивний SCORM 1.2 тренажер для практичної роботи з теми аналізу залежностей між змінними.

## Структура

- `src/index.html` — головна сторінка тренажера
- `src/style.css` — стилі
- `src/imsmanifest.xml` — SCORM-маніфест
